## 30 天退款到底从哪天算起？

先把这个问题说死：**从账户注册那天起算，不是从你下单买 VPS 那天算起。**

举个最典型的坑：你注册了账号，但搬瓦工限量套餐经常缺货，于是等了两周才抢到，用了半个月觉得线路不合适——这时候从购买日算还有两周，但账号注册已经超过 30 天了，退款直接被拒。

所以如果你现在注册了账号还没买，**窗口已经在倒计时**。买了之后要尽快评估，不是慢慢等。

## 搬瓦工退款的 8 个条件，一条没满足就会被拒

搬瓦工的退款不是"无理由退款"，是有门槛的。拿着下面这张清单自查，全部打钩才能申请：

| 条件 | 说明 |
|---|---|
| ① 账户状态正常 | 没有欠款，未违反服务条款 |
| ② 无争议记录 | 没有发起过 PayPal 争议或银行 Chargeback |
| ③ IP 干净 | 分配的 IP 未被黑名单收录，且没有申请过免费换 IP |
| ④ 流量使用 < 10% | 当月数据传输未超配额的 10% |
| ⑤ 服务数量 ≤ 3 | 账户下 VPS 总数（含已取消）不超过 3 个 |
| ⑥ 总付款 ≤ $60 | 历史付款总额不超过 60 美元，次数不超过 10 次 |
| ⑦ 账户年龄 ≤ 30 天 | 注册时间不超过 30 天（最关键） |
| ⑧ 首次退款 | 未曾在其他账户申请过退款 |

说实话，前 6 条大多数人都能满足，最容易被刷掉的是第 7 条——注册时间一过，什么都晚了。

第 4 条也要注意。测试阶段如果跑了大量数据（比如传文件、跑脚本压测），流量很容易超 10%，这时候退款就没戏。30 天窗口期里，功能测试尽量轻量，别把数据全量迁过去再说"不合适"。

## 两种退款申请方式

### 方式一：自动退款系统（推荐）

登录账号，访问搬瓦工官方退款申请页面。系统会自动检测你是否满足退款条件，满足的话直接提交，之后会收到确认邮件，**回复 "I agree"** 即可。整个流程从提交到收到退款确认，一般几十分钟到几小时内完成。

### 方式二：提交工单

登录后台提交 Support Ticket，Subject 填 "Refund Request"，正文说明想退款即可。等人工处理，周期稍长一些。

这两种方式选哪个都行，不过有一点特别重要：**绝对不要通过 PayPal 发起争议（Dispute）来逼退款**。搬瓦工的服务条款写得很清楚，发起 Chargeback 或 PayPal Dispute 会直接导致名下及关联账户所有 VPS 立即暂停，后果很难收拾。

## 退款之后，数据怎么办？

一旦退款批准，账户下**所有 VPS 立即关停，所有数据、快照、备份全部永久删除，不可恢复**。

申请前务必把需要保留的东西全部备份到本地或其他地方。这不是可以事后找回来的——真的不行。

另外，退款是全额退款，不按使用天数折算。买了年付套餐，用了 29 天申请退款，钱全退，一分不少。这点是正经的。

## 退款后账号还能用吗？

可以用，账号本身不会注销。但是，**每个账号只有一次退款机会**，退完之后这个账号不再享有退款权利。

所以如果你退了之后想换个套餐重新买，同一个账号继续用就行，只是这次买了就没有退款兜底了。

## 退款常见失败原因

值得提一下几个高频踩坑场景：

**IP 被封/换过 IP 的情况**：搬瓦工里有些使用场景容易触发 IP 被封，如果你的 VPS IP 出现过黑名单收录，或者用过免费换 IP 功能，退款条件里的第 ③ 条就不满足了。申请之前先在 KiwiVM 后台确认 IP 状态。

**账户超过 30 天**：如上所说，买之前先确认注册日期，别拖太久才去测试。

**流量超标**：跑了高强度测试或者大量下载，这个月流量超过配额 10%，就不符合退款条件了。

**总金额超过 $60**：如果你在同一账户下买了多个套餐，或者买了月付香港 CN2 GIA 这种价格较高的套餐，总金额可能超过 $60 上限。自动系统会拒绝，但用工单提申请的话，实际上有不少用户反映人工审核通过了——可以先试试提工单。

## FAQ

**Q：我买了年付，只用了几天，能全额退款吗？**

A：满足退款条件的话，是的，全额退，不按天数折算。搬瓦工的退款是按账户从注册起的所有付款退，不是按剩余使用时间比例计算。

**Q：退款可以退到账户余额，下次再用吗？**

A：不行。退款只能退回原始支付渠道——支付宝退支付宝，PayPal 退 PayPal，信用卡退信用卡。

**Q：我的账号注册超过 30 天了，但 VPS 是刚买的，能退吗？**

A：不能通过自动系统退。账号注册时间超过 30 天这条是硬条件，系统会直接拒绝。可以提工单说明情况，但通过的概率不高，建议买之前先确认账号注册日期。

**Q：退款后账号还能继续用吗？还能再买吗？**

A：账号不会注销，可以继续购买其他套餐。但这个账号不再享有退款权利，之后买了就不能再退了。

**Q：退款审批了之后，我的数据还能找回来吗？**

A：不能。退款批准的瞬间，所有 VPS、数据、快照、备份全部立即永久删除。申请前务必先备份。
